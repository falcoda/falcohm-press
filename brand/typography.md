# Typographie

**Inter** uniquement (la police du site Falc'ohm). Cinq graisses chargées dans `kit/core.py`.

| Alias | Fichier | Usage |
|---|---|---|
| `Inter-B` | Inter-Bold | Titres de page, chiffres clés, kickers |
| `Inter-SB` | Inter-SemiBold | Titres de cartes, nom du contact |
| `Inter-M` | Inter-Medium | Eyebrows, légendes, pastilles |
| `Inter` | Inter-Regular | (peu utilisé) |
| `Inter-L` | Inter-Light | Texte courant, sous-titres, gros titres en contraste |

## Échelle
| Élément | Taille | Interligne |
|---|---|---|
| Titre couverture | 44 pt | — |
| Titre de page | 28-30 pt | ×1,16 |
| Chapô | 11-13 pt | 17-20,5 |
| Texte courant | 9,6 pt | 15,4 |
| Titre de carte | 10,2 pt | 13,4 |
| Corps de carte | 8,6 pt | 12,8 |
| Légende / pied | 7-8 pt | — |

**Règle :** un titre en `Inter-B` + une ligne en `Inter-L` de même corps = le contraste maison.
Ne pas introduire d'italique ni de police de titrage supplémentaire.
