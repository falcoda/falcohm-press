# Grille et espacements

- Format A4 : 595,28 × 841,89 pt
- Marge `M` = 52 pt sur les quatre côtés (colonne utile `CW` = 491,28 pt)
- Filet de pied de page à y = 56 ; aucun contenu sous y = 84
- Gouttière entre cartes : 12 pt ; entre blocs de l'écosystème : 34 pt
- Rayon des cartes : 8 pt (6 pt pour les mockups)
- Bandes photo : toujours pleine largeur (bord à bord), jamais dans la marge

## Rythme vertical d'une page de contenu
```
y = H - 86        kicker
y -= 34           titre
y -= 30/32        chapô
...               contenu
y >= 84           dernier élément
```
Un espace vide en bas de page est préférable à une page saturée.
