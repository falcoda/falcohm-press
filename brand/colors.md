# Couleurs

Définies dans `kit/core.py`. Ne jamais coder une couleur en dur dans une page.

| Rôle | Hex | Usage |
|---|---|---|
| `BLACK` | `#0E0F12` | Fonds pleine page (couverture, page finale) |
| `INK` | `#191B20` | Titres, bandeaux sombres |
| `BODY` | `#4B5058` | Texte courant |
| `GREY` | `#8A9099` | Texte secondaire, légendes |
| `LIGHT` | `#F3F4F6` | Fonds de page alternés, cartes teintées |
| `LINE` | `#E3E5E9` | Filets, contours de cartes |
| `BLUE` | `#1F5FEB` | Accent unique : icônes, kickers, bandeaux, pastilles |
| `WHITE` | `#FFFFFF` | Fonds, texte sur fond sombre |

**Règle :** un seul accent (le bleu). Pas de seconde couleur d'accent, pas de dégradé de couleur.
Les dégradés existants sont uniquement des voiles noirs sur photo (`veil`).
